公司需要一个面向客户的门户，让客户能够下载固件、产品文档、许可证等内容，因此需在 Azure 中构建一个高可靠、可扩展、可全球访问的解决方案。

### 贡献（Contribution）

- 在 **Azure Container Apps** 上部署完整后端服务，实现无服务器自动扩缩容

- 使用 **Azure Blob Storage** 管理固件、PDF 文档、License 文件  
- 通过 GitLab CI/CD 实现容器镜像构建、推送与自动发布  
- 使用 Azure Monitor 提供日志、指标和监控告警  
- 设计整个平台架构，使其安全、自动化、可扩展且运维成本低

### 价值（Value Created）
